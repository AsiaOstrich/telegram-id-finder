# Telegram ID Finder
# Telegram ID 查詢工具

Web UI tool to find Telegram group/channel IDs and user IDs for configuration.
網頁工具：取得 Telegram 群組、頻道、用戶 ID。

---

## Features | 功能

- Find Telegram group IDs | 查詢 Telegram 群組 ID
- Find Telegram channel IDs | 查詢 Telegram 頻道 ID
- Find Telegram user IDs | 查詢 Telegram 用戶 ID
- Web-based interface | 網頁介面
